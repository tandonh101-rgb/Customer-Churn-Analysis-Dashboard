# 📊 Customer Churn Analysis Dashboard (Power BI)
 📝 Project Overview
The Customer Churn Analysis Dashboard is an interactive Power BI project designed to analyze customer behavior and identify patterns that lead to churn. This dashboard helps businesses understand why customers leave and provides insights to improve customer retention strategies.

 🎯 Objectives

* Identify key factors contributing to customer churn
* Analyze customer demographics and behavior
* Track churn trends over time
* Provide actionable insights for business decision-making

 📂 Dataset Description

The dataset includes the following key columns:

* Customer_ID – Unique identifier for each customer
* Gender – Male/Female
* Age – Customer age
* Marital Status – Married/Unmarried
* State – Customer location
* Tenure – Number of months with the company
* Services Used – Phone, Internet, Multiple Lines, etc.
* Contract Type – Monthly/Yearly
* Payment Method – Mode of payment
* Monthly Charges & Total Charges
* Churn – Indicates whether the customer left (Yes/No)

🛠️ Tools & Technologies

* Power BI – Data visualization and dashboard creation
* SQL – Data cleaning and preprocessing
* Python – Data analysis (NumPy, Pandas)

 🔄 Data Cleaning Steps

* Handled missing values (NULL replacement)
* Removed duplicates
* Standardized categorical values (e.g., Yes/No)
* Converted data types (e.g., text to numeric)
* Created calculated columns and measures where required

 📊 Dashboard Features

* KPIs Section

  * Total Customers
  * Total Churned Customers
  * Churn Rate (%)

* Visualizations

  * Churn by Gender
  * Churn by Age Group
  * Churn by Contract Type
  * Monthly Charges vs Churn
  * Tenure Distribution
  * State-wise Churn Analysis

* Filters / Slicers

  * Gender
  * Contract Type
  * Payment Method
  * State

## 📈 Key Insights

* Customers with monthly contracts have higher churn rates
* Higher monthly charges are linked to increased churn
* Customers with short tenure are more likely to leave
* Certain regions/states show higher churn patterns
